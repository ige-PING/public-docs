
# LMDZ
The LMDZ model is used at IGE. The code and the documentation are managed at LMD (Jussieu, Paris), within a [website](https://lmdz.lmd.jussieu.fr/).

This model is typically run on IPSL plateforms ([IPSL mesocentre](https://mesocentre.ipsl.fr/)), or at [Idris](http://www.idris.fr/) or at [TGCC](https://www-hpc.cea.fr/fr/TGCC.html), trough computing projects managed within [GENCI](https://www.genci.fr/) (Gerhard Krinner is managing a computing time demands for LMDZ at IGE).

A limited-area version of this model is developed, you can join the [mattermost chanel](https://mattermost.lmd.ipsl.fr/lmdz/channels/sos-lam) if you want to join  this initiative.
