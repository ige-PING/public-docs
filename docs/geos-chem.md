# GEOS-Chem 

GEOS-Chem is a global 3-D model of atmospheric chemistry driven by meteorological input from the Goddard Earth Observing System (GEOS) of the NASA Global Modeling and Assimilation Office. 

Ressources :
  - [Guide for running GEOS-Chem on IGE computing resources](https://github.com/GEOS-Chem-IGE/GEOS-Chem-infra)
  - [Official GEOS-Chem website](https://geoschem.github.io/index.html)
  - [Official GEOS-Chem Documentation](https://geos-chem.readthedocs.io/en/latest/index.html)
  - [GEOS-Chem source code](https://github.com/geoschem)
