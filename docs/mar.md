# MAR

MAR is a regional climate model. From July 2025, the MAR model is hosted at https://gitlab.uliege.be/especes/mar/stable. Specific IGE documentation is hosted at [https://gitlab.com/Mar-Group/MARv3](https://gitlab.com/Mar-Group/MARv3). This page contains the model code until the MARv3.14.2 version as well as a general documentation (how to install, prepare your setup, and run the model).
Instuctions specific to Dahu can be found in the issues and in the [wiki part](https://gitlab.com/Mar-Group/MARv3/-/wikis/home), with a specific page for [outputs](https://gitlab.com/Mar-Group/MARv3/-/wikis/Outputs).
